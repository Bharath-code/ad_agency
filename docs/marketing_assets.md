# Marketing Assets Checklist — AI Visibility Intelligence

> All assets needed for a successful launch.

---

## Pre-Launch Assets

### Brand Identity

- [ ] **Logo files**
  - [ ] Primary logo (SVG, PNG @2x)
  - [ ] Icon only (SVG, PNG @2x)
  - [ ] Wordmark only (SVG, PNG @2x)
  - [ ] Dark mode variants
  - [ ] Favicon (ICO, 16x16, 32x32)

- [ ] **Color palette** (in Figma/design tool)
- [ ] **Typography specimens** (Inter, JetBrains Mono)

---

### Website Assets

- [ ] **Open Graph image** (1200×630)
- [ ] **Twitter card image** (1200×600)
- [ ] **Apple touch icon** (180×180)

### Product Screenshots

- [ ] **Dashboard overview** (clean state with real data)

## Launch Day Assets

### Product Hunt

- [ ] **Tagline** (60 chars max): "See how AI sees you"
- [ ] **Description** (260 chars max)
- [ ] **Gallery images** (5-8 images, 1270×760 each)
- [ ] **Maker story** (optional video or comment)
- [ ] **First comment** (launch context)

### Hacker News

- [ ] **Show HN title** (<80 chars)
- [ ] **First comment** (what it does, why built)

### Twitter/X

- [ ] **Launch thread** (5-7 tweets)
- [ ] **Animated GIF** of product in action
- [ ] **Quote cards** for key stats

### LinkedIn

- [ ] **Launch post** (text + image)
- [ ] **Founder story angle**
