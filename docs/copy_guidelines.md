## Formatting Rules

### Capitalization

| Element | Style | Example |
|---------|-------|---------|
| Headlines | Sentence case | "See how AI sees you" |
| Buttons | Sentence case | "Run free scan" |
| Labels | Sentence case | "Visibility score" |
| Product names | Proper case | "AI Visibility Intelligence" |

### Punctuation

- Headlines: No period
- Body text: Regular punctuation
- Lists: No period on single items
- CTAs: No period

### Contractions

Use them. They're friendlier.
- "You're" not "You are"
- "It's" not "It is"
- "Don't" not "Do not"
