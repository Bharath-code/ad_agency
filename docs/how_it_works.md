## Under the Hood (Technical, Simple)

### What We Actually Do

```
1. SEND QUERY TO AI
   "What's the best project management tool?"
   
2. GET RESPONSE
   "I recommend Notion, Linear, and Asana..."
   
3. PARSE RESPONSE
   - Find brand mentions
   - Detect position (1st, 2nd, not mentioned)
   - Extract reasoning context
   
4. COMPARE TO COMPETITORS
   - Who's mentioned when you're not?
   - What phrases/features do they have?
   
5. GENERATE RECOMMENDATIONS
   - Pattern match: "Winners mention X, you don't"
   - Suggest specific copy/content changes
   
6. CALCULATE SCORE
   - % of queries where you appear
   - Weighted by query importance
```

No. We analyze patterns and help you improve. We don't manipulate AI.

Weekly scans + weekly email reports. You can always run a manual scan in the dashboard.
